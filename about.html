About the Project

Project Overview

This system is a State-of-the-art Multi-Modal AI solution designed to combat modern email threats. Traditional filters often miss spam hidden in images or screenshots; our system bridges this gap by merging text and visual intelligence.

Safe (Ham) Suspicious Spam

How It Works

The system utilizes a Late-Fusion Multimodal Architecture:

  • DistilBERT: Lightweight Transformer for Text Analysis.
  • MobileNetV2: Efficient CNN for Image detection & OCR processing.

Decision Logic

  • SPAM: Triggered if both models flag the content.
  • SUSPICIOUS: Triggered if either model detects a threat.
  • HAM: Confirmed safe when both models clear the input.
